### 2. How is Data prepared for Machine Learning by AltexSoft
" The data should be prepared in such a way that the ML model should clearly understand what should be done using the data set"
- The amount of data depends on the complexity of the project.
- The quality of the selected data set is very important to make ML model understand that what work it has to carryout.
- Data Preparation : It is one of the important procedure in the process of making the ML model to understand and to carry out the work. It Includes :
   - Labelling : The ML model should be provided with the features of data set so that it can identify the various things in data set easily and accurately.This could be done by providing the ML model with more examples.
   - Mislabelled Data Samples : During labelling some samples can be mislabelled leading to wrong classification of items.
   - Reduction and Cleansing : The data should be reduced to as small as it can be and only the required data samples should be chosen from the data set because large data require more time to process and more dimensions makes algorithms to not work properly that is we should use dimensionality reducing process while creating data.
   - Sampling : It should be used to balance the imbalancing data set.
   - Data Wrangling : It is a process in which the data is expressed in such a way that normalizes the given problem.
   - Formatting : It should be used to format  what is not needed and keeping the needed.
   - Feature Engineering : The conversion of historical data into normalized data and also adding new features to carry out work.

## Objective
The objective of this task was to understand the working of a 555 timer IC in astable mode and design a circuit that generates a square wave with a duty cycle of approximately 60%. The task also aimed to observe the output waveform using a Digital Storage Oscilloscope (DSO).
